# Alex Chen — Creative Developer Portfolio

A premium editorial-style developer portfolio featuring refined minimal aesthetics with distinctive typography and subtle interactions.

## Framework

Static HTML with Vanilla JavaScript

## Design Direction

**Refined Minimal Editorial** — Inspired by high-end design publications with:
- Sophisticated serif headings (Crimson Pro) paired with clean sans-serif body (Inter)
- Generous whitespace and deliberate typography hierarchy
- Monochromatic palette with strategic accent color
- Subtle motion: hover lifts, staggered reveals, parallax hero
- **Unforgettable detail**: Custom cursor with blend mode that expands on hover

## Theme

Clean light theme with editorial sophistication:
- Paper white backgrounds (#fafafa)
- Ink black text (#1a1a1a)
- Strategic blue accent (#2563eb)
- Soft shadows and refined borders
- Premium, calm, intentional

## Features

- Fixed navigation with backdrop blur
- Hero section with staggered reveal animations
- Social links with directional hover states
- Projects grid with image hover effects and lift animations
- Skills breakdown by category
- Visual timeline with gradient line
- High-contrast contact card
- Custom cursor with blend mode interaction
- Smooth scroll and parallax effects
